# excel--projects
Project Overview

This project is a data analysis dashboard built using dataset from the Texas Department of Insurance.

The dataset consists of approximately 2,73,750+ records with around 9–11 columns, collected from data.texas.gov.

🔍 Data Cleaning & Preprocessing

- Relevant columns were selected for meaningful analysis.
- Missing or blank values were handled by replacing them with "Unknown" to maintain data integrity.
- Date fields were standardized into YMD (Year-Month-Day) format.

📈 Dashboard Insights

The dashboard provides visualization and analysis of:

- 📌 Complaint volume trend over multiple years
- 📌 Top complaint categories and frequently reported issues
- 📌 Average complaint resolution time by complaint type
- 📌 Comparison between individual and organizational complaints
- 📌 Coverage-wise complaint distribution
- 📌 Complaint filing source activity analysis

🎯 Project Impact

The dashboard helps in identifying complaint trends, operational performance, and service improvement opportunities through interactive visual insights.
